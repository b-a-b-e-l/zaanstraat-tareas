# Zaanstraat 314 · Tareas

Página de seguimiento de tareas para la compra y remodelación de Zaanstraat 314, Amsterdam.

## Archivos

```
index.html   ← página principal
style.css    ← estilos
app.js       ← lógica: idiomas, progreso, localStorage
```

## Subir a GitHub Pages (gratis)

1. Crear un repositorio en GitHub (ej. `zaanstraat-tareas` o en el mismo repo `b-a-b-e-l`)
2. Subir los tres archivos: `index.html`, `style.css`, `app.js`
3. Ir a **Settings → Pages → Source → main branch / root**
4. En un minuto la página estará en: `https://TU-USER.github.io/zaanstraat-tareas/`
